Uma operação é uma característica comportamental que pode ser de propriedade de uma interface ou classe. Ela pode ser diretamente invocada em instâncias da classe que a contém. Uma operação especifica o nome, tipo, parâmetros e restrições para tais invocações.

Uma operação abstrata é uma que não tem implementação, ou seja, a classe não a implementa. Isso faz com que um descendente desta mesma classe deva fazê-lo.
