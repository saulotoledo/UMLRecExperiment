Muitas linguagems, a exemplo de C++, tem a noção de classes template (também chamadas de parametrizadas). Com elas, você pode utilizar a definição geral para fazer uso de elementos mais específicos. Um conjunto, por exemplo, pode ser definido como uma classe parametrizada e, durante seu uso, o tipo do conjunto pode ser espeficicamente definido.
