Uma associação define um relacionamento entre duas entidades conceituais. Ela define responsabilidades entre duas classes.

Navegabilidade, em uma associação, é um termo utilizado para descrever a direção para a qual se direciona um relacionamento. Um objeto que está contido dentro de outro objeto é dito ser navegável se o objeto continente possui métodos para acessar o objeto contido. A maioria dos relacionamentos são navegáveis em uma direção, mas se ambos os objetos contiverem referências entre si, é possível se ter um relacionamento de navegabilidade bidirecional.
