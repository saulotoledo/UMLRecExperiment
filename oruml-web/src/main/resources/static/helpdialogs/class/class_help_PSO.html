Uma operação é uma característica comportamental que pode ser de propriedade de uma interface ou classe. Uma operação pode ser diretamente invocada em instâncias da classe que a contém. Uma operação especifica o nome, tipo, parâmetros e restrições para tais invocações.

Uma operação é estática se pertence à classe, e não a suas instâncias. Assim, uma operação com essa característica deve ser acessado através da classe que a contém de maneira estática, e não a partir de suas instâncias.
