Atributos são elementos que definem a estrutura de uma classe.

Um atributo estático é um atributo que pertence à classe, e não a suas instâncias. Assim, um atributo com essa característica deve ser acessado através da classe que o contém de maneira estática, e não a partir de suas instâncias.
