Uma associação define um relacionamento entre duas entidades conceituais. Ela define responsabilidades entre duas classes.

Uma agregação é um caso especial de associação que modela um relacionamento tem um (ou parte de, no jargão da UML) entre pares. Esta agregação é partilhada (outrora conhecida apenas como "agregação" em versões anteriores da UML) quando os objetos participantes podem existir independentemente uns dos outros. Nenhum objeto é mais importante do que o outro no relacionamento.
