O principal conteúdo em um diagrama de sequência é o conjunto de mensagens. Uma mensagem é apresentada por uma seta de uma linha de vida para outra. A seta aponta para o destinatário. Se a mensagem é assíncrona, a linha tem uma seta fina. Uma mensagem assíncrona não requer uma resposta antes que o remetente continuar. Uma mensagem assíncrona mostra apenas uma chamada do remetente. Use para representar a comunicação entre segmentos separados ou a criação de um novo segmento.
