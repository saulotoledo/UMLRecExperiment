Fragmento combinado representado pela tag par. Este fragmento indica que ocorre paralelismo entre os fluxos contidos em seus operandos. Cada sub-região representa uma computação paralela (concorrente). Na maioria dos casos, cada sub-região contém diferentes linhas de vida. A execução das mensagens em cada sub-região é sequencial, mas a ordem relativa das mensagens em sub-regiões paralelas é completamente arbitrária.
