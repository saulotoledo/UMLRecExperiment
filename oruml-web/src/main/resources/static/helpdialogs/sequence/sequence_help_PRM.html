O principal conteúdo em um diagrama de sequência é o conjunto de mensagens. Uma mensagem é apresentada por uma seta de uma linha de vida para outra. A seta aponta para o destinatário. A resposta a uma mensagem síncrona (um retorno de chamada) é exibida por uma linha tracejada com uma seta fina. A mensagem de retorno pode ser omitida, já que há um retorno implícito após qualquer chamada, mas frequentemente é útil mostrar os valores de retorno.
